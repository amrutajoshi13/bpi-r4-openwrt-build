# BPI-R4 OpenWrt Build Environment


## Requirements
- Ubuntu 18.04 or later
- Docker installed
- 50GB free disk space
- 8GB RAM minimum


## Quick Start
git clone git@gitlab.com:yourname/bpi-r4-openwrt-build.git
cd bpi-r4-openwrt-build
docker build -t openwrt-bpir4 .
